# /archive - Archive Completed Embedded System Change

Archive a completed change by merging delta specifications into main specs.

## Supported Spec Types

- hardware/
- firmware/
- protocols/
- power-management/

## Archive Process

Merges ADDED/MODIFIED/REMOVED specifications from `specs/changes/<change-id>/` into the main specification directories, then moves the change to the archive.
